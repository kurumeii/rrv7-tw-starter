# Task Completion Checklist

Before considering a task complete, verify the following:

1.  **Formatting & Linting:** Run `pnpm lint` to ensure code style adherence and fix any issues.
2.  **Type Checking:** Run `pnpm typecheck` to ensure no TypeScript errors.
3.  **Spell Check:** Run `pnpm spellcheck` to catch typo errors.
4.  **Verification:** Test the implemented feature/fix in the development environment (`pnpm dev`).
5.  **Build Check:** Run `pnpm build` to ensure the application builds successfully for production.
