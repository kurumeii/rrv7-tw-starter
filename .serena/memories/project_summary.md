# Project Summary
**Project:** rrv7-tw-starter
**Description:** A starter project for React Router v7 using Tailwind CSS v4, Remix Themes, and React Aria Components.
**System:** Linux

## Tech Stack
*   **Framework:** React Router v7 (SSR enabled)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS v4, `tailwind-variants` (for component variants), `tailwind-merge`
*   **UI Components:** `react-aria-components` (RAC)
*   **Icons:** `@remixicon/react`
*   **Theme Management:** `remix-themes` (Dark/Light mode)
*   **Tooling:** Vite, Biome, CSpell, PostCSS, Ladle
*   **Package Manager:** pnpm

## Key Directories
*   `app/routes/`: Route handler files
*   `app/components/ui/`: Reusable UI components (RAC-based)
*   `app/configs/`: Configuration files
*   `app/server/`: Server-side utilities
*   `app/routes.ts`: Manual route configuration
*   `app/root.tsx`: Root layout component
