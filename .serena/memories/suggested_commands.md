# Suggested Commands

## Development
*   `pnpm dev`: Start the development server.
*   `pnpm ladle`: Start Ladle server for component development/stories.
*   `pnpm start`: Start the production server (after build).

## Build & Test
*   `pnpm build`: Build the project for production.
*   `pnpm typecheck`: Run TypeScript type checking.
*   `pnpm ladle:build`: Build Ladle static site.

## Code Quality
*   `pnpm lint`: Run Biome to check and fix linting and formatting issues (`biome check --write app`).
*   `pnpm spellcheck`: Run CSpell on the `app` directory.

## Utility
*   `pnpm install`: Install dependencies.
